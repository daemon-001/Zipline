# Zipline - File Transfer Application Requirements
# This file lists the system and development requirements for Zipline

# =============================================================================
# SYSTEM REQUIREMENTS
# =============================================================================

# Operating System
- Windows 10/11 (64-bit)
- macOS 10.14+ (Mojave or later)
- Linux (Ubuntu 18.04+ or equivalent)

# Hardware
- RAM: 512 MB minimum, 2 GB recommended
- Storage: 100 MB free space
- Network: Local area network (LAN) connection
- CPU: x64 architecture

# =============================================================================
# DEVELOPMENT REQUIREMENTS
# =============================================================================

# Core Development Tools
Flutter SDK >= 3.9.2
Dart SDK >= 3.9.2
Git >= 2.0.0

# IDE (Choose one)
- Visual Studio Code with Flutter extension
- Android Studio with Flutter plugin
- IntelliJ IDEA with Flutter plugin

# Platform-Specific Requirements

# Windows
- Visual Studio 2022 Build Tools
- Windows 10 SDK (10.0.19041.0 or later)
- Microsoft Visual C++ Redistributable

# macOS
- Xcode 12.0 or later
- CocoaPods
- Command Line Tools for Xcode

# Linux
- clang
- cmake
- ninja-build
- pkg-config
- libgtk-3-dev
- libblkid-dev
- liblzma-dev

# =============================================================================
# BUILD REQUIREMENTS
# =============================================================================

# For Windows builds
- Visual Studio 2022 Build Tools
- Windows 10 SDK
- CMake 3.16 or later

# For macOS builds
- Xcode 12.0+
- CocoaPods
- macOS 10.14+

# For Linux builds
- clang
- cmake
- ninja-build
- pkg-config
- GTK3 development libraries

# =============================================================================
# RUNTIME REQUIREMENTS
# =============================================================================

# Network
- Local area network (LAN) connection
- UDP/TCP port 7250 (configurable)
- Multicast support for network discovery

# File System
- Read/write access to selected directories
- Sufficient disk space for file transfers
- Support for long file paths (Windows)

# Permissions
- Network access
- File system access
- System tray access (optional)

# =============================================================================
# INSTALLATION INSTRUCTIONS
# =============================================================================

# 1. Install Flutter SDK
#    Download from: https://flutter.dev/docs/get-started/install

# 2. Install platform-specific tools
#    Windows: Visual Studio 2022 Build Tools
#    macOS: Xcode and CocoaPods
#    Linux: clang, cmake, ninja-build, pkg-config

# 3. Clone repository
#    git clone https://github.com/yourusername/zipline.git

# 4. Install dependencies
#    cd zipline
#    flutter pub get

# 5. Build application
#    flutter build windows    # for Windows
#    flutter build macos      # for macOS
#    flutter build linux      # for Linux

# =============================================================================
# TROUBLESHOOTING
# =============================================================================

# Common Issues:
# - Flutter doctor: Run 'flutter doctor' to check setup
# - Network discovery: Ensure devices are on same network
# - Firewall: Allow Zipline through Windows Firewall
# - Port conflicts: Change port in settings if 7250 is in use
# - Build errors: Run 'flutter clean && flutter pub get'
